# Election Management System (EMS) 🗳️

A robust Java-based application designed to digitize the electoral process. This project showcases **Full-Stack Development** skills using Java and MySQL, tailored for enterprise-level reliability.

## 🚀 Key Features
* **Voter Authentication:** Secure login for voters and administrators.
* **Voter Database:** Complete CRUD operations for voter registration.
* **Candidate Tracking:** Manage and display candidate profiles.
* **Automated Tallying:** Instant and accurate vote counting to eliminate human error.

## 🛠 Tech Stack
* **Language:** Java (JDK 8+)
* **GUI:** Java Swing / AWT
* **Database:** MySQL (connected via JDBC)
* **IDE:** NetBeans / Eclipse

## 📂 Installation
1. Clone the repo: `git clone https://github.com/harshsakhare/Election_Management_System.git`
2. Import into your Java IDE.
3. Import the database schema from the `/electionsdb` folder.
4. Run `LOGIN.java` to start the app.
